Projects:
TPCH Cassandra Ingest
A Clojure application where we ingest a TPCH generated database into Cassandra.
Tic Tac Toe Cubed
A real-time, turn based game which is an interesting variation of the age old game of Tic Tac Toe. Built on the NodeJS platform using express framework and the source can be found at https://github.com/dhruvasagar/tictactoecubed

VIM Table Mode
VIM plugin offering a table mode for easy table creation, manipulation and auto formatting, inspired by Emacs Org Modes tables. Also includes the ability to define formulas similar to excel enabling complex calculations across range of cells.
VIM Powerless Line
VIM statusline plugin inspired by VIM Powerline.
VIM Markify
VIM plugin to indicate quickfix / location list results on buffers using VIM Signs.
MM Versionable
MM Versionable is a plugin for MongoMapper a data mapper style ruby library for MongoDB. MM Versionable adds the ability to version documents for changes so they can be rollbacked at any time to any state.
Soulmate Rails
Extends Soulmate to build a Rails plugin for easy integration with Rails Models. Soulmate is a library offering auto completion for words, built using Redis sorted sets.
Technology Stack: Rails

Hash Calculator
Hash Calculator is a simple Java Swing based GUI application that helps in calculating various hashes (MD2, MD5, SHA-1, SHA-256, SHA-384, SHA-512) of text / files.
