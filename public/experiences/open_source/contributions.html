Contributions:
VIM Railscasts Theme
VIM colorscheme railscasts forked and added terminal support (256 colors).
QuickTask
QuickTask is a comprehensive Task Management plugin for VIM.
Powerline
Powerline is a plugin that provides fancy statuslines for a wide variety of tools like VIM, tmux, shells, etc.
SuperModel
SuperModel is a Ruby library providing simple in-memory models. Very useful for testing Rails plugins without actually needing to setup & configure a database. I used this for testing Soulmate Rails gem.
Technology Stack: Ruby
Lint.vim
Lint.vim is a VIM plugin provides VIM integration with JSHint & CSSLint.
VIM Task
Vim Task is a VIM plugin for very simple task management.
VIM Ruby Doc
VIM Ruby Doc is a VIM plugin that enables easy & quick Ruby Doc lookups from within VIM.
VIM jQuery Doc
VIM jQuery Doc is a VIM plugin that enables easy & quick jQuery Doc lookups from within VIM.
VIM JST
VIM JST is a VIM plugin that adds JST support to VIM.
